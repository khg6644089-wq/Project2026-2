## 1. 사이트 개요

- **라스트밸런스**는 식단·운동·커뮤니티를 한 곳에서 관리하는 웹 서비스입니다.
- **주요 기능**: 식단 사진 분석(칼로리), 운동 기록, 식단/식품 관리, 클럽(커뮤니티), 이벤트(포인트샵), 챗봇 등.

---

<a id="toc-2"></a>
## 2. 접속 및 회원

### 2.1 접속

- 브라우저에서 사이트 주소로 접속합니다.
- **메인(홈)으로 이동**: 상단 **로고**를 클릭합니다.

### 2.2 로그인 / 로그아웃

- **로그인**: 상단 메뉴에서 **로그인**을 클릭한 뒤, 이메일과 비밀번호를 입력하고 로그인합니다.
- **로그아웃**: 로그인 후 상단 메뉴에서 **로그아웃**을 클릭합니다.

### 2.3 회원가입

- 상단 메뉴에서 **로그인**을 클릭해 로그인 화면으로 이동한 뒤, 화면 안의 **회원가입** 링크를 클릭합니다. 필요한 정보를 입력하고 가입을 완료합니다.

---

<a id="toc-3"></a>
## 3. 홈(메인)

**이동 방법**: 상단 **로고**를 클릭하면 메인 화면으로 이동합니다.

### 3.1 메인 화면

- **배너 슬라이드**: 이벤트 배너가 슬라이드로 표시됩니다. 배너를 클릭하면 해당 이벤트 상세 화면으로 이동합니다.
- **오늘 먹은 음식은 몇 칼로리일까요?**
  - **음식 사진 업로드**: "클릭해서 음식 사진 업로드" 버튼을 눌러 이미지를 선택합니다.
  - **분석 결과**: 업로드 후 칼로리, 음식명, 평가 문구가 자동으로 표시됩니다.
  - **음식이 감지되지 않을 때**: "사진에서 음식이 없습니다." 메시지가 나오면, 음식이 잘 보이는 사진으로 다시 올려 주세요.
- **추천 커뮤니티**: 최신순 클럽 목록이 카드로 보입니다. 카드를 클릭하면 해당 클럽 상세 화면으로 이동합니다. (상단 메뉴 **커뮤니티**에서도 전체 목록을 볼 수 있습니다.)

<a id="toc-4"></a>
## 4. 라스트밸런스 소개

**이동 방법**: 상단 메뉴에서 **라스트밸런스**를 클릭합니다.

- 식단 관리, 운동 기록, 커뮤니티 등 서비스 소개 내용을 확인할 수 있습니다.

<a id="toc-5"></a>
## 5. 커뮤니티

**이동 방법**: 상단 메뉴에서 **커뮤니티**를 클릭합니다.

### 5.1 커뮤니티 목록

- **정렬**: 최신순, 회원 많은 순, 게시글 많은 순 등으로 목록을 바꿀 수 있습니다.
- **검색**: 키워드로 클럽을 검색할 수 있습니다.
- **클럽 카드**: 클럽명, 설명, 이미지, 태그가 표시됩니다. 카드를 클릭하면 해당 클럽 상세 화면으로 이동합니다.

### 5.2 클럽 상세

**이동 방법**: 커뮤니티 목록에서 원하는 **클럽 카드**를 클릭합니다.

- **클럽 정보**: 이름, 설명, 이미지, 태그, 회원 수 등을 볼 수 있습니다.
- **가입 신청**: 아직 가입하지 않은 경우 "가입 신청" 버튼으로 신청합니다. (승인 방식이면 관리자 승인 후 가입이 완료됩니다.)
- **게시판**: 가입한 클럽에서는 게시글 목록을 볼 수 있습니다.
- **글쓰기**: 가입 회원은 "글쓰기"로 게시글을 작성할 수 있습니다.
- **게시글 보기**: 제목을 클릭하면 글 내용, 댓글, 좋아요를 확인하고 댓글을 달 수 있습니다.

<a id="toc-6"></a>
## 6. 이벤트

**이동 방법**: 상단 메뉴에서 **이벤트**를 클릭합니다.

### 6.1 이벤트 화면

- **포인트샵**: 포인트로 응모할 수 있는 상품 목록이 표시됩니다.
- **내 포인트**: 로그인한 경우 보유 포인트가 표시됩니다.
- **상품 응모**: 로그인한 뒤 원하는 상품의 "응모" 버튼을 누르면 포인트가 차감되고 응모가 완료됩니다. (포인트가 부족하면 응모할 수 없습니다.)
- **이벤트 상세**: 배너나 상품을 클릭하면 해당 이벤트/상품 상세 화면으로 이동합니다.

<a id="toc-7"></a>
## 7. 밸런스 체크(마이페이지)

**이동 방법**: 상단 메뉴에서 **밸런스 체크** 또는 **(로그인 시) ○○님의 밸런스 체크**를 클릭합니다.

> 밸런스 체크 메뉴는 **로그인한 사용자만** 이용할 수 있습니다. 비로그인 시 클릭하면 로그인 안내가 나옵니다.

### 7.1 밸런스 체크 메인(대시보드)

**이동 방법**: 상단 메뉴에서 **밸런스 체크**(또는 **○○님의 밸런스 체크**)를 클릭하면 처음 들어오는 화면입니다.

- **주간 체중·식단·운동 요약**: 차트와 요약으로 한 주간 기록을 볼 수 있습니다.
- **내가 가입한 클럽**: 가입한 커뮤니티 목록을 볼 수 있습니다.
- **내가 쓴 글**: 내가 작성한 게시글 목록을 볼 수 있습니다.
- 화면 안의 링크/버튼으로 **식단 기록**, **운동 기록**, **식품 관리**, **내 정보**, **주간 기록**, **CM 기록** 등으로 이동할 수 있습니다.

### 7.2 식단 기록(식사 기록)

**이동 방법**: 밸런스 체크 메인 화면에서 **식단 기록** 메뉴(또는 해당 버튼)를 클릭합니다.

- 날짜별 식단(식사) 기록 목록을 볼 수 있습니다.
- 식단 추가·수정·삭제가 가능합니다.

### 7.3 식단 작성

**이동 방법**: 밸런스 체크 내 식단 관련 화면에서 **식단 작성**(또는 글쓰기) 버튼을 클릭합니다.

- 새 식사 기록을 작성합니다. 식사 유형, 음식, 칼로리 등을 입력할 수 있습니다.

### 7.4 운동 기록

**이동 방법**: 밸런스 체크 메인 화면에서 **운동 기록** 메뉴(또는 해당 버튼)를 클릭합니다.

- 날짜별 운동 기록 목록을 볼 수 있습니다.
- **날짜 선택**: 캘린더나 날짜 선택으로 해당 날짜의 기록을 봅니다.
- **기록 삭제**: 특정 운동 기록을 삭제할 수 있습니다.

### 7.5 운동 기록 작성

**이동 방법**: 운동 기록 화면에서 **운동 기록 작성**(또는 작성/추가) 버튼을 클릭합니다.

- **날짜 선택**: 기록할 날짜를 선택합니다.
- **운동별 시간 입력**: 코어, 유산소, 근력, 유연성, HIIT 등 카테고리별로 운동 종목과 시간(분)을 입력합니다.
- **계산**: "계산" 버튼을 누르면 소모 칼로리가 계산됩니다.
- **저장**: "저장" 버튼으로 해당 날짜의 운동 기록을 저장합니다.

### 7.6 식품 관리

**이동 방법**: 밸런스 체크 메인 화면에서 **식품 관리** 메뉴(또는 해당 버튼)를 클릭합니다.

- 자주 먹는 식품·재료를 등록·수정·삭제하여, 식단 기록할 때 빠르게 고를 수 있도록 관리합니다.

### 7.7 내 정보

**이동 방법**: 밸런스 체크 메인 화면에서 **내 정보** 메뉴(또는 해당 버튼)를 클릭합니다.

- **프로필**: 이름, 성별, 키, 몸무게 등 회원 정보를 확인하고 수정할 수 있습니다.
- **비밀번호 변경**: 비밀번호 변경 링크(또는 버튼)를 누르면 비밀번호 변경 화면으로 이동합니다.

### 7.8 주간 기록

**이동 방법**: 밸런스 체크 메인 화면에서 **주간 기록** 메뉴(또는 해당 버튼)를 클릭합니다.

- 주간 단위 체중·식단·운동 등 기록을 조회하고 관리할 수 있습니다.

### 7.9 CM 기록

**이동 방법**: 밸런스 체크 메인 화면에서 **CM 기록** 메뉴(또는 해당 버튼)를 클릭합니다.

- 커뮤니티 매니저(CM) 활동과 관련된 기록을 볼 수 있습니다. (해당 권한이 있는 경우)

## 8. 커뮤니티 관리

**이동 방법**: 상단 메뉴에서 **커뮤니티 관리**를 클릭합니다.

> **권한**: 내가 **클럽 매니저(운영자)**인 경우에만 이 메뉴가 보이고 들어갈 수 있습니다.

### 8.1 커뮤니티 관리 메인

**이동 방법**: 상단 메뉴에서 **커뮤니티 관리**를 클릭하면 처음 나오는 화면입니다.

- **내가 관리하는 클럽**: 내가 매니저인 클럽 목록이 표시됩니다.
- **프로필 요약**: 이름, 성별, 키/몸무게, 프로필 이미지 등이 보일 수 있습니다.
- 화면 안의 메뉴/버튼으로 **커뮤니티 열기**, **가입 신청 목록**, **공지 관리** 등으로 이동할 수 있습니다.

### 8.2 커뮤니티 열기(클럽 운영)

**이동 방법**: 커뮤니티 관리 메인에서 **커뮤니티 열기**(또는 해당 클럽)를 클릭합니다.

- 관리하는 클럽을 선택해 클럽 상세, 게시판, 공지 등을 관리하는 화면입니다.

### 8.3 가입 신청 목록

**이동 방법**: 커뮤니티 관리 메인에서 **가입 신청 목록** 메뉴(또는 버튼)를 클릭합니다.

- 클럽에 대한 **가입 신청 목록**을 볼 수 있습니다.
- **승인/거절**: 각 신청에 대해 승인 또는 거절할 수 있습니다.

### 8.4 공지 관리

**이동 방법**: 커뮤니티 관리 메인에서 **공지 관리** 메뉴(또는 버튼)를 클릭합니다.

- 클럽 **공지사항 목록**을 보고, 수정·삭제할 수 있습니다.
- **공지 작성**: "공지 작성" 등 버튼을 누르면 새 공지를 작성하는 화면으로 이동합니다.
  - **제목**, **내용**을 입력하고, 공지를 올릴 **클럽**을 선택한 뒤 저장하면 해당 클럽 공지로 등록됩니다.

## 9. 어드민

**이동 방법**: 상단 메뉴에서 **어드민**을 클릭합니다.

> **권한**: **사이트 관리자** 권한이 있는 계정으로 로그인한 경우에만 이 메뉴가 보이고 이용할 수 있습니다.

### 9.1 어드민 메인(대시보드)

**이동 방법**: 상단 메뉴에서 **어드민**을 클릭하면 처음 나오는 화면입니다.

- 사이트 전체 현황(회원 수, 클럽 수 등) 요약을 볼 수 있습니다.

### 9.2 회원 관리

**이동 방법**: 어드민 메인(또는 어드민 좌측 메뉴)에서 **회원 관리**를 클릭합니다.

- 전체 회원 목록을 조회하고, 필요 시 회원을 정지(제재)할 수 있습니다.

### 9.3 클럽 회원 관리

**이동 방법**: 어드민 메인(또는 어드민 좌측 메뉴)에서 **클럽 회원 관리**를 클릭합니다.

- 클럽별 회원 목록을 보고, 특정 클럽에서 회원을 제거할 수 있습니다.

### 9.4 클럽 관리

**이동 방법**: 어드민 메인(또는 어드민 좌측 메뉴)에서 **클럽 관리**를 클릭합니다.

- 전체 클럽 목록을 보고, 클럽을 비활성화(정지)할 수 있습니다.

### 9.5 클럽 상세(어드민)

**이동 방법**: 클럽 관리 목록에서 해당 **클럽**을 클릭합니다.

- 해당 클럽의 상세 정보와 통계를 보고, 필요한 관리 작업을 할 수 있습니다.

<a id="toc-10"></a>
## 10. 챗봇

- **보이는 경우**: **로그인한 사용자**에게만 화면 우측 하단 등에 챗봇 버튼이 표시됩니다.
- **사용 방법**: 챗봇 버튼을 클릭하면 채팅 패널이 열립니다. 메시지를 입력해 대화할 수 있습니다.
- **닫기**: 패널의 닫기 버튼을 누르면 챗봇 창이 닫힙니다.

| 메뉴 | 이동 방법 | 비고 |
|------|------------|------|

| 로그인 / 로그아웃 | 상단 **로그인** 또는 **로그아웃** 클릭 | 회원가입은 로그인 화면에서 |

| 커뮤니티 관리 | 상단 **커뮤니티 관리** 클릭 | 클럽 매니저만 표시 |

## 자주 묻는 질문(FAQ)

- **비로그인 상태에서 밸런스 체크 메뉴를 누르면?**  
  "로그인 시 이용 가능합니다" 안내가 나오며, 로그인을 하면 이용할 수 있습니다.

- **음식 사진을 올렸는데 "사진에서 음식이 없습니다"가 나와요.**  
  음식이 선명하게 보이는 사진으로 다시 촬영하거나 업로드해 주세요.

- **커뮤니티 관리 / 어드민 메뉴가 안 보여요.**  
  커뮤니티 관리는 **클럽 매니저**, 어드민은 **사이트 관리자** 권한이 있는 계정에서만 보입니다. 해당 권한이 없으면 메뉴가 표시되지 않습니다.

- **이벤트에서 "포인트가 부족합니다"가 나와요.**  
  보유 포인트가 상품 응모에 필요한 포인트보다 적습니다. 포인트를 더 적립한 뒤 다시 시도해 주세요.
