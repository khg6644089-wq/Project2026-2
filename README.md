### 문제 인식

현재 헬스 기록 서비스는 대부분 개인 중심이거나 단순 정보 기록에 그치는 경우가 많아, 사용자가 꾸준히 건강 관리 습관을 유지하기 어렵습니다. 또한 건강 관리는 의지만으로 지속하기 힘들어 시작조차 어렵고,
수많은 다이어트·식단 관리·헬스 케어 앱 중 어떤 서비스를 선택해야 할지 판단하기 어려운 상황입니다.
이러한 문제로 인해 사용자는 건강 목표를 달성하기 어렵고, 서비스 제공자 입장에서도 장기적인 참여를 유도하기 어렵습니다.

### 해결 방안

기존 헬스 기록 서비스는 꾸준한 건강 관리와 서비스 선택에서 어려움을 겪습니다. 
라스트 밸런스는 커뮤니티 기반 공유, 꾸준한 기록과 보상, AI 맞춤형 지원을 통해 사용자가 지속적으로 운동과 식단을 관리하도록 돕고, 건강한 습관 형성과 목표 달성을 동시에 지원합니다.

## 📋 주요 기능



### 1. 로그인

- 로그인하지 않은 사용자는 **메인, 소개, 이벤트 메뉴**만 이용 가능
- 다른 기능 클릭 시 **로그인 페이지로 이동**
- 회원이 아닐 경우, 로그인 페이지에서 **회원가입 페이지로 이동 가능한 버튼** 제공
- 로그인 성공 시 **메인 페이지로 이동**, 메뉴에 **사용자 이름 표시** (예: "홍지승님 안녕하세요")

### 2. 메인페이지(HOME)

- 이벤트 내용 4개
- 기능 소개
- 음식 사진 넣으면 칼로리 계산됨
- 금주 신설된 최신 모임 4개

### 3. 커뮤니티
- 커뮤니티 전체 리스트 확인 가능
- 페이지네이션으로 이동
- 커뮤니티 검색 기능 존재
- 포인트를 모아 CM이 되면 모임 개설 가능
- 어드민 계정의 경우 **커뮤니티 정지 버튼** 노출
  - 클릭 시 해당 커뮤니티 정지
  - 커뮤니티 이용 정지

### 4. 이벤트 페이지

식단 인증
- 30일 동안 식단 인증 업로드 시 → **스포애니 50% 할인권** 지급
운동 인증
- 운동 인증 30회 성공 시 → **랩노쉬 단백질바 5개** 증정
목표 체중 달성
- 목표 체중 달성 시 → **백화점 상품권 10만원** 지급

### 5. 마이페이지

개인 정보 및 목표
- 목표 체중, 보유 포인트 확인
- 이름, 비밀번호, 휴대폰, 키, 몸무게, 알레르기, 성별, 특이사항 **수정 가능**
  
식단 관리
- 사진으로 칼로리 자동 계산 및 기록
- 하루 섭취 칼로리 누적 계산 및 목표와 비교
- 추천 식단 확인 (매일 다른 식단, 성분 분석, 코멘트 제공)
- 주간 작성 기록 조회
  
운동 기록
- 하루 운동량 기록 및 지난주 비교
- 운동 완료 시 **오운완 표시**
- 일주일간 체중 변화 기록 (월요일 업데이트)

커뮤니티 활동
- 작성한 글 내역 리스트 확인
- 내용 많을 시 페이지네이션 적용

### 6. 문의사항 (챗봇)

- **로그인 필수**
- 우측 하단 플로팅 버튼으로 상시 접근 가능
- AI 챗봇을 통해 **체력관리, 식단 관리, 운동 추천** 관련 질문 가능
  - 각 버튼 클릭 시 해당 카테고리 안내 문구 제공 (예: "체력 관리에 대해서 질문해주세요!")
- 사용자의 질문 입력 창 제공
- 논문 기반 학습으로 답변 제공
- 범위 외 질문 시 "답변이 불가합니다" 안내

### 7. CM 메뉴

메뉴 접근
- CM 유저 로그인 시 메뉴바에 표시
- CM은 일반 유저 메뉴도 이용 가능

커뮤니티 관리
- **커뮤니티 개설**
  - 이름, 소개, 키워드 5개, 배경 이미지 입력
  - 미입력 시 자동으로 사용자 이름 사용
  - CM은 1개 커뮤니티만 개설 가능
  - 개설 완료 시 내가 CM으로 있는 커뮤니티 페이지에 노출
- **가입 신청 관리**
  - 알림 수신 및 팝업/페이지에서 수락·거절 가능
- **공지사항 관리**
  - 작성된 공지사항 리스트 확인 (번호, 제목, 작성자, 작성일자)
  - 페이지네이션 적용
  - 공지사항 작성 가능
    - 글 제목, 내용, 이미지 1개 첨부
    - 작성 완료 → 게시글 페이지 이동
    - 작성 취소 → 공지사항 관리 페이지 이동

### 8. 어드민

 **회원 정지**
  - 악플러, 상인 등 불량 계정 정지 가능
  - 커뮤니티 게시글에서 삭제 버튼 클릭 → 계정 정지
  - 삭제 시 확인 팝업 (Y/N)

- **커뮤니티 정지**
  - 커뮤니티 자체 이용 정지 버튼 클릭 → 해당 커뮤니티 이용 정지

- **알림**
  - 이용 정지된 계정 로그인 시 → "이용 정지된 회원입니다" 알림
  - 정지된 커뮤니티 클릭 시 → "정지된 커뮤니티 입니다" 알림
